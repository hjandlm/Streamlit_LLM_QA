## Streamlit_LLM_QA

### 项目介绍
本项目使用streamlit集成多个大模型+外部知识检索进行AI问答。

本项目支持模型选择，是否使用外部知识，选择外部知识检索方式，模型参数设置等方面。

代码详细介绍请查看：https://mp.weixin.qq.com/s/P7JvmjNrumzIgpjOHGOSoQ
